Details About API

GET - /publicApi/users

Returns an array of all the users in the database.

GET - /publicApi/users/:user_id

Returns a specific user.

GET - /publicApi/groups

Returns an array of all the groups in the database.

GET - /publicApi/group/:group_id

Returns a specific group.

GET - /publicApi/apiuser

Returns your current profile, more detailed than using findUser.

GET - /publicApi/apiuser/groups

Returns your groups, more detailed than using findGroup.
